Animation!

Keyframes

Flere tilstander

@keyframes move {
  from { translateX(0); }
  to { translateX(100px); }
}

from og to er synonymer for 0% og 100%

@keyframes move {
  0% { translateX(0); }
  100% { translateX(100px); }
}

Kan inferere 0% fra css

thing {
  translateX(0);
}
@keyframes move {
  100% { translateX(100px); }
}

En animasjon med kun to tilstander er essensielt en transition

men animasjoner kan ha flere tilstander!

@keyframes move {
  0% { translateX(0); }
  10% { translateX(100px); }
  100% { translateX(200px); }
}

Ikke begrenset til kun en property per tilstand

@keyframes roll {
  0% {
    transform: translateX(0) rotate(0);
    background-color: #f00;
  }
  30% {
    transform: translateX(100px) rotate(1turn);
    background-color: #bada55;
  }
  100% {
    transform: translateX(200px) rotate(2turn);
    background-color: #ff69b4;
  }
}

Kan gi et element flere animasjoner som med transitions

Flere properties for tuning

  • iteration-count
  • direction
  • fill-mode

normal, reverse, alternate, alternate-verse

animation-direction: normal

number, infinite

none, forwards, backwards, both
