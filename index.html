Hva er flex?

Dynamisk Layout Modul for CSS

Flex er en layout modul for CSS som gir en mer effektiv måte å lage fleksible / dynamiske layout hvor man ikke kjenner størrelsene på alle elementene.

Fordeler luft

Du spesifiserer størrelsen på elementene, hvordan de skal plasseres og så lar du pcen gjøre resten.

flex-grow: (default 0)

Flex-grow spesifiserer man forholdet et element kan vokse hvis det får lov i forhold til de andre

flex-shrink: (default 1*)

  • 1

*0 for IE 10/11 ლ(ಠ益ಠლ)

Shrink forklarer hvor liten ett element kan bli relativt i forhold til de andre. Med grow og shrink kan du da lage elementer som automatisk vokser og krymper for å tilpasse seg i forhold til hverandre
